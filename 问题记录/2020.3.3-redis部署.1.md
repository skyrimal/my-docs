# linux 安装redis

## 5.0.7

[安装教程](https://blog.csdn.net/Weixiaohuai/article/details/103429412)

在要解压位置用mkdis 创建redis文件夹

cd 跳到该文件夹

tar -zxvf redis-5.0.7.tar.gz 解压redis安装压缩包

可以用mv命令将文件夹转移 用空格将源文件夹和目标文件夹分开

> 解压到usr/local目录

安装gcc-c++

进入redis解压后的文件夹，执行make命令

进入redis下src文件夹，执行make install命令

安装完成

在redis安装目录下执行./src/redis-server ./redis.conf 启动redis

> MISCONF Redis is configured to save RDB snapshots, but it is currently not able to persist on disk
>
> 百度翻译： missf Redis已配置为保存RDB快照，但它当前无法在磁盘上持久化。
>
> 解决：1--启动redis-cli，执行 127.0.0.1:6379> config set stop-writes-on-bgsave-error no
>
> 解决：2--vi打开redis-server配置的redis.conf文件，然后使用快捷匹配模式：/stop-writes-on-bgsave-error定位到stop-writes-on-bgsave-error字符串所在位置，接着把后面的yes设置为no即可。
